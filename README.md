# Sensor Flight Data

This dataset contains environmental and motion data recorded over time during a descent from ~109 m altitude to ground level.

## Columns

| Column | Description | Unit |
|------|-------------|------|
| Time_s | Timestamp since start of measurement | seconds |
| Temperature_C | Ambient temperature | °C |
| Pressure_Pa | Atmospheric pressure | Pascals |
| Altitude_m | Calculated altitude | meters |
| AX_m_s2 | Acceleration on X axis | m/s² |
| AY_m_s2 | Acceleration on Y axis | m/s² |
| AZ_m_s2 | Acceleration on Z axis | m/s² |
| UV_raw | Raw UV sensor reading | sensor units |
| ALS_raw | Ambient light sensor reading | sensor units |
| CO2_ppm | Carbon dioxide concentration | ppm |
